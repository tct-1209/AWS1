# Worklog Week 1: AWS Free Tier, IAM, and AWS CLI Foundations

**Implementation period:** 20/04/2026 - 26/04/2026

## 1. Weekly objectives

- Understand AWS Free Tier, credit management, and common cost risks.
- Study IAM users, groups, policies, roles, and the least-privilege principle.
- Install AWS CLI, configure profiles, and run basic account-checking commands.
