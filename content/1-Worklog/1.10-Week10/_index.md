# Worklog Week 10: Data, NoSQL, Analytics, and Database Migration

**Implementation period:** 22/06/2026 - 28/06/2026

## 1. Weekly objectives

- Study Data Lake, Glue, Athena, and QuickSight.
- Understand DynamoDB and NoSQL data modeling.
- Review schema conversion and database migration to AWS.

## 2. Implementation details

| Day | Date | Work details |
| --- | --- | --- |
| Monday | 22/06/2026 | Studied Data Lake architecture and the role of S3 in raw-data storage. |
| Tuesday | 23/06/2026 | Reviewed AWS Glue Crawler/Data Catalog for preparing data for queries. |
| Wednesday | 24/06/2026 | Studied Amazon Athena queries and QuickSight visualization. |
| Thursday | 25/06/2026 | Reviewed DynamoDB, partition keys, sort keys, and NoSQL table design. |
| Friday | 26/06/2026 | Studied schema conversion and database migration workflows in real systems. |

## 3. Outcomes

- Understood how data is collected, cataloged, analyzed, and visualized on AWS.
- Learned the basic difference between relational databases and NoSQL.
- Gained perspective on data migration during system modernization.
