# Worklog Week 6: Application Containerization with Docker, ECR, and ECS

**Implementation period:** 25/05/2026 - 31/05/2026

## 1. Weekly objectives

- Understand Docker containers and the value of containerization.
- Study Amazon ECR, ECS Cluster, Task Definition, Service, and Load Balancer.
- Understand CI/CD concepts for containerized applications.

## 2. Implementation details

| Day | Date | Work details |
| --- | --- | --- |
| Monday | 25/05/2026 | Studied Docker images, containers, Dockerfiles, and Docker Compose in a local environment. |
| Tuesday | 26/05/2026 | Reviewed how to push images to Amazon ECR and deploy applications from images. |
| Wednesday | 27/05/2026 | Studied ECS Cluster, Task Definition, and how ECS manages containers. |
| Thursday | 28/05/2026 | Reviewed Application Load Balancer configuration for ECS services. |
| Friday | 29/05/2026 | Studied CI/CD models with GitHub Actions, GitLab Runner, and CodeBuild for containers. |

## 3. Outcomes

- Understood the process of packaging applications with Docker.
- Learned the key components of deploying containers on ECS.
- Recognized how CI/CD reduces manual deployment work.
