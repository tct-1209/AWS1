# Worklog Tuần 6: Container hóa ứng dụng với Docker, ECR và ECS

**Thời gian thực hiện:** 25/05/2026 - 31/05/2026

## 1. Mục tiêu tuần

- Hiểu Docker container và lợi ích của container hóa.
- Tìm hiểu Amazon ECR, ECS Cluster, Task Definition, Service và Load Balancer.
- Nắm khái niệm CI/CD cho ứng dụng container.

## 2. Chi tiết công việc triển khai

| Thứ | Thời gian | Nội dung công việc |
| --- | --- | --- |
| Thứ 2 | 25/05/2026 | Tìm hiểu Docker image, container, Dockerfile và Docker Compose ở môi trường local. |
| Thứ 3 | 26/05/2026 | Nghiên cứu cách push image lên Amazon ECR và sử dụng image để deploy. |
| Thứ 4 | 27/05/2026 | Tìm hiểu ECS Cluster, Task Definition và cách ECS quản lý container. |
| Thứ 5 | 28/05/2026 | Xem quy trình cấu hình Application Load Balancer cho service chạy trên ECS. |
| Thứ 6 | 29/05/2026 | Đọc mô hình CI/CD với GitHub Actions, GitLab Runner và CodeBuild cho container. |

## 3. Kết quả đạt được

- Hiểu quy trình đóng gói ứng dụng bằng Docker.
- Nắm các thành phần chính khi triển khai container trên ECS.
- Biết CI/CD giúp giảm thao tác deploy thủ công.
