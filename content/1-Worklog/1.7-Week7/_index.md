# Worklog Week 7: Application Security and Access Governance

**Implementation period:** 01/06/2026 - 07/06/2026

## 1. Weekly objectives

- Study AWS WAF and web application protection.
- Understand the role of Security Hub, KMS, and IAM conditions in security.
- Practice access-control thinking through tags and roles.

## 2. Implementation details

| Day | Date | Work details |
| --- | --- | --- |
| Monday | 01/06/2026 | Studied AWS WAF, Web ACLs, managed rules, and custom rules for web applications. |
| Tuesday | 02/06/2026 | Reviewed Security Hub to understand centralized security findings and monitoring. |
| Wednesday | 03/06/2026 | Studied encryption at rest with AWS KMS and key-management principles. |
| Thursday | 04/06/2026 | Reviewed IAM Role & Condition for condition-based access control. |

- Created a security checklist for the internship project.
