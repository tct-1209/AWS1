### Speaker sessions

#### 1. Bao Huynh - Docker: A containerization technology

This session helped me understand Docker as a containerization technology that packages applications with their runtime environment, making deployment more consistent across local machines, servers, and CI/CD pipelines.

#### 2. Le Hoang Gia Dai - Combining AWS WAF with Machine Learning for Cyber Attack Detection

The speaker shared how AWS WAF can be combined with Machine Learning to detect abnormal traffic and improve web application protection. I learned that security can be strengthened through data and analytical models.

#### 3. Nguyen Quoc Bao - Multiplayer in the Cloud: Connecting Godot Clients with AWS WebSockets

This presentation focused on connecting Godot game clients to a cloud backend through WebSockets. It helped me better understand realtime applications, low-latency communication, and client-server architecture on AWS.

#### 4. Truong - Technical sharing session

This session helped me observe how a technical topic can be analyzed, presented, and connected with practical use cases. It was useful for improving technical explanation and problem-analysis skills.

#### 5. Viet Phat - AWS Neptune for Building a Graph Knowledge Base for GraphRAG

The speaker introduced Amazon Neptune and graph knowledge bases for GraphRAG. I learned more about graph databases, entity relationships, and the role of knowledge graphs in AI-assisted retrieval.

#### 6. Vinh Tran - From IT Helpdesk to Senior Sysadmin

This career-oriented session emphasized self-learning, experience documentation, system fundamentals, and the process of gradually developing professional skills in system administration.

### Outcomes

Event 2 expanded my knowledge of Docker, web application security, WebSockets, GraphRAG, and career development in system operations. I also learned how to present technical topics in a concise, clear, and practical way.
