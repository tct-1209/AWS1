# Event 3: Cloud Architect Final & Technical Sharing

### Event information

| Item | Information |
|---|---|
| **Event name** | Cloud Architect Final & Technical Sharing |
| **Time** | 09:00, 11/07/2026 |
| **Location** | 26th Floor, Bitexco Tower, 02 Hai Trieu Street, Saigon Ward, Ho Chi Minh City |
| **Role** | Attendee |

### Opening session

The event opened with the **Cloud Architect** final round between **KLKAT** and **Ngu Dai Hiep**. This competition helped me observe how teams analyzed cloud architecture problems, presented their solutions, and defended their ideas in front of the audience.

### Speaker sessions

#### 1. Thinh Nguyen - Securing Your Web Apps With AWS Security Agent

This session focused on protecting web applications deployed on AWS. I gained a better understanding of security layers, risk reduction, and the importance of monitoring and early issue detection during operations.

#### 2. Nguyen Huynh Son - SLA and Monitoring: From SLA to Monitoring What Really Matters

The speaker shared insights about SLA and how to monitor metrics that truly matter. I learned that monitoring is not only about dashboards, but also about user experience, system reliability, and operational goals.

#### 3. Ngo Le Tan Huy - Inside The Exam: AWS Cloud Practitioner

This session provided an overview of the AWS Cloud Practitioner exam, including how to prepare foundational knowledge, manage study time, and review effectively. It motivated me to continue learning toward AWS certification.

### Outcomes

After the event, I gained a clearer understanding of cloud architecture thinking, web application security, goal-oriented monitoring, and AWS certification preparation. The event also helped me improve my ability to observe, take notes, and summarize technical sharing sessions.
